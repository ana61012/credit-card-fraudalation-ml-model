# Credit Card Fraud Detection

## Overview
Built a fraud detection system using Random Forest on an imbalanced dataset of 284,807 transactions.

## Dataset
- PCA-transformed features (V1–V28)
- Transaction Amount
- Binary target: Fraud (1) / Valid (0)

## Techniques Used
- Stratified train-test split
- Class balancing
- ROC-AUC evaluation
- Confusion matrix analysis
- Feature importance visualization

## Results
- High recall for fraud detection
- Strong ROC-AUC performance
